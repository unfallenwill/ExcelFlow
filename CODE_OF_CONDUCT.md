# 社区行为准则

我们致力于为所有参与者提供友善、尊重且无骚扰的协作环境，不因经验、身份、背景或观点而区别对待。

参与项目时请：

- 聚焦事实、代码和用户需求，进行建设性讨论；
- 尊重不同经验水平，耐心解释并接受善意反馈；
- 不公开他人的私人信息或项目中的敏感业务数据；
- 不使用侮辱、威胁、歧视、骚扰或人身攻击；
- 在分歧升级时暂停讨论并请求维护者协助。

维护者可以编辑或移除不当内容，并根据严重程度警告、暂时限制或永久禁止参与。需要报告行为问题时，请先通过[仓库所有者的 GitHub 主页](https://github.com/unfallenwill)请求建立私密沟通，不要在公开讨论中披露事件细节或受影响者的私人信息。本行为准则适用于项目仓库及维护者代表项目参与的其他社区空间。
