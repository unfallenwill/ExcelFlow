# 安全政策

## 支持版本

项目目前处于早期阶段，仅承诺评估最新发布版本中的安全问题；旧版本不会获得单独维护。建议用户及时升级。这个说明是维护策略，不构成响应时间或修复时限承诺。

## 报告安全问题

请不要在公开 Issue 中披露可利用细节、真实业务数据、凭据或个人信息。优先使用 GitHub 仓库的 [私密安全报告入口](https://github.com/unfallenwill/ExcelFlow/security/advisories/new)。如果仓库未启用该入口，请先通过仓库所有者的 GitHub 主页联系，并且只提供建立私密沟通所需的最少信息；不要在公开资料中附上漏洞细节。

报告应包括受影响版本、复现条件、潜在影响和建议缓解方式。维护者确认后会协调修复和披露时间。

ExcelFlow 会读取用户提供的 Excel，并在本地处理数据。衍生列表达式使用受限解释器，不应允许任意 Python。发现绕过限制的方式应按安全问题私下报告。
